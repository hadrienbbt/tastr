# Tastr
Native WebApp to meet new people

## How to execute
### Step 1 :
`git clone https://github.com/hadrienbbt/tastr`
`cd tastr`

### Step 2 :
Install mongoDB
`mongod --dbpath server/db`

### Step 4 :
Install node.js
In another terminal window, type:
`npm install`
`cd server`
`node app.js`

### Step 4 :
In another terminal window, type:
`cd tastr/tastr`
`react-native link`
`react-native run-ios`
